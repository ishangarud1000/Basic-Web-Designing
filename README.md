# Web Dev Playground: Building Blocks of the Modern Web

This repository serves as a collection of projects and exercises focused on front-end web development. It primarily features bascic of code built using the foundational trio: HTML, CSS, and JavaScript.

## Key Technologies

HTML5: Structure and content for all projects.

CSS3: Styling, layout, and visual design (including Flexbox, Grid, and responsive design techniques).

JavaScript (ES6+): Interactivity, DOM manipulation, and application logic

### What You'll Find
Mini-Projects: Small, focused applications demonstrating specific techniques (e.g., a modal, a calculator, a clock).
